# IronPDF C++ Landing Page

## Project Overview
This project is a pixel-perfect implementation of the provided Figma design for the **IronPDF C++ Landing Page**.

It is built using **CodeIgniter 4** and follows a clean MVC structure. Dynamic content is loaded from a JSON file to simulate real-world data handling.

---

## How to Run the Project

### 1. Clone the repository
```bash
git clone https://github.com/pyaesoneag877/ironpdf-cpp-landing-page
cd ironpdf-cpp-landing-page
2. Start PHP development server
php -S localhost:8080 -t public
3. Open in browser
http://localhost:8080
